  • VS Code插件

    • CSS Peek：快速查看CSS定义

    • IntelliSense for CSS：CSS智能提示

    • PostCSS Language Support：PostCSS支持
    • Color Highlight：颜色高亮

  • Sublime Text插件

    • CSS3：CSS3语法支持

  • WebStorm/PhpStorm

    • 内置CSS支持
    • 实时预览
    • 代码补全
    • 重构工具

# 2. 构建工具

  • PostCSS插件

    // postcss.config.js
module.exports = {
  plugins: [
    require('autoprefixer'),
    require('postcss-preset-env'),
    require('cssnano'),
    require('postcss-import'),
    require('postcss-nested')
  ]
}

  • Webpack配置

    // webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/,
        use: [
          'style-loader',
          'css-loader',
          'postcss-loader'
        ]
      }
    ]
  }
}

  • Vite配置

    // vite.config.js
export default {
  css: {
    preprocessorOptions: {
      scss: {
        additionalData: `@import "@/styles/variables.scss";`
      }
    }
  }
}

# 2. 调试技巧

# 使用边框调试

/* 为元素添加边框 */
.debug {
  border: 1px solid red;
}

/* 为特定元素添加边框 */
.debug * {
  border: 1px solid blue;
}

/* 使用不同颜色区分 */
.debug-parent {
  border: 2px solid red;
}

.debug-child {
  border: 2px solid blue;
}

# 使用outline调试

/* 使用outline不会影响布局 */
.debug-outline {
  outline: 2px solid red;
}

/* 为所有元素添加outline */
.debug-outline * {
  outline: 1px solid blue;
}

# 使用背景色调试

/* 使用半透明背景色 */
.debug-bg {
  background-color: rgba(255, 0, 0, 0.1);
}

/* 使用不同背景色区分层级 */
.debug-bg * {
  background-color: rgba(0, 0, 255, 0.1);
}
