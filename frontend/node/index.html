# 技术选型

# 为什么选择 Node.js？

  1. 技术优势

    • 异步非阻塞 I/O
    • 事件驱动模型
    • 丰富的 npm 生态
    • 前后端统一语言

  2. 适用场景

    • Web 应用开发
    • 实时应用
    • 微服务架构
    • 工具开发

  3. 开发效率

    • 快速迭代
    • 代码复用
    • 丰富的框架
    • 活跃的社区

# 技术栈选择

  1. Web 框架

    • Express.js - 轻量级框架
    • Koa.js - 现代化框架
    • Nest.js - 企业级框架
    • Fastify - 高性能框架

  2. 数据库

    • MySQL/PostgreSQL - 关系型数据库
    • MongoDB - 文档数据库
    • Redis - 缓存数据库
    • Elasticsearch - 搜索引擎

  3. 开发工具

    • npm/yarn/pnpm - 包管理
    • ESLint - 代码检查
    • Jest/Mocha - 测试框架
    • PM2 - 进程管理

# 快速开始

# 环境准备

  1. 安装 Node.js

    # 使用 nvm 安装
nvm install node

# 或直接下载安装包
# 访问 https://nodejs.org

  2. 验证安装

    node --version
npm --version

# 创建项目

# 创建项目目录
mkdir my-node-app
cd my-node-app

# 初始化项目
npm init -y

# 安装依赖
npm install express

# 开发调试

# 启动开发服务器
npm run dev

# 使用 nodemon 实现热重载
npm install nodemon --save-dev
