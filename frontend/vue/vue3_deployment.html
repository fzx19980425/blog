# Vue 3 项目部署

Vue 3 推荐使用 Vite 作为构建工具，支持更快的开发和构建体验。这里对照 Vue2，介绍 Vue3 的部署流程。

# 1. 构建配置

# 1.1 Vite 配置

// vite.config.js
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
  base: '/',
  build: {
    outDir: 'dist',
    assetsDir: 'static',
    sourcemap: false
  }
})

# 1.2 环境变量

# .env.production
VITE_API_BASE_URL=/api
