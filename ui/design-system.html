# 阴影系统

# 阴影层级

  • 基础阴影：0 2px 4px rgba(0, 0, 0, .12), 0 0 6px rgba(0, 0, 0, .04)
  • 卡片阴影：0 2px 12px 0 rgba(0, 0, 0, 0.1)
  • 弹窗阴影：0 2px 12px 0 rgba(0, 0, 0, 0.1)

# 应用场景

  • 卡片：卡片阴影
  • 弹窗：弹窗阴影
  • 下拉菜单：基础阴影
  • 悬浮按钮：基础阴影

# 动画系统

# 动画时长

  • 快速：0.2s
  • 基础：0.3s
  • 慢速：0.4s

# 动画曲线

  • 标准：cubic-bezier(0.4, 0, 0.2, 1)
  • 加速：cubic-bezier(0.4, 0, 1, 1)
  • 减速：cubic-bezier(0, 0, 0.2, 1)

# 应用场景

  • 按钮点击：快速 + 标准
  • 页面切换：基础 + 标准
  • 弹窗显示：基础 + 标准
  • 提示信息：快速 + 标准

# 响应式断点

# 断点定义

  • 超小屏幕：< 768px
  • 小屏幕：≥ 768px
  • 中等屏幕：≥ 992px
  • 大屏幕：≥ 1200px
  • 超大屏幕：≥ 1920px

# 应用建议

  • 移动优先设计
  • 使用相对单位（rem/em）
  • 弹性布局
  • 媒体查询适配
